# ShieldSign Brand Assets

Place your brand assets in this directory.

## Required Files

### Logos
- `logo.svg` - Full logo (recommended: 200x50px)
- `logo-dark.svg` - Logo for dark backgrounds
- `logo-icon.svg` - Icon only (square, 64x64px min)
- `logo-icon-dark.svg` - Icon for dark backgrounds

### Favicon
- `favicon.ico` - Multi-size favicon (16, 32, 48px)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` - 180x180px

### OpenGraph / Social
- `og-image.png` - 1200x630px for social sharing
- `twitter-card.png` - 1200x600px for Twitter

## Color Specifications

**Primary Color:** #0a3b3c (Deep Teal)
**Accent Color:** #14b8a6 (Emerald/Light Teal)

## Usage

After adding assets:
1. Copy to `apps/remix/public/`
2. Update `packages/assets/site.webmanifest`
3. Rebuild application
